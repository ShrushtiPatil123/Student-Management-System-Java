# Student Management System

## Overview

The Student Management System is a Java-based web application developed to manage student records efficiently. It provides complete CRUD (Create, Read, Update, Delete) functionality through a user-friendly interface. The application is built using JSP, Servlets, JDBC, MySQL, Maven, and Apache Tomcat following the DAO design pattern.

## Features

- Admin Login
- Add Student
- View Students
- Update Student
- Delete Student
- MySQL Database Integration
- Responsive User Interface
- CRUD Operations

## Technologies Used

## How to Run

1. Clone the repository.
2. Import the project as a Maven Project.
3. Configure MySQL database.
4. Update database credentials in `DBConnection.java`.
5. Run the project using Apache Tomcat.
6. Open:
http://localhost:8080/StudentManagementSystem
